## When to use this component

This component is a standard code implementation for displaying easy to read dates to users from [ISO format](https://en.wikipedia.org/wiki/ISO_8601).

Follow the [GOV.UK date guidance](https://www.gov.uk/guidance/style-guide/a-to-z-of-gov-uk-style#dates) to decide which format to use within your service.
