Help users to

Access a service with security codes

Also known as authenticate or identifying users with two factor authentication

EXPERIMENTAL

This pattern is currently experimental because more research is needed to validate it.

Let users access a service in a secure way by authenticating themselves with security codes.

When to use this pattern

You can use security codes or two-factor authentication to verify a users identity to let them access a service. Two-factor authentication can be used when a service stores user data and allows users to access it. Two-factor helps protect the user and the service.

There are three types of two-factor authentication: something a user knows (password, pin), something a user has (phone, fob), and a biometric like a fingerprint or voice.

Services using this pattern

  • Access UK
  • Apply for the EU Settlement Scheme
  • Status: right to rent, right to work, view & prove
  • Update your EU Settlement Scheme details

Research

More research is needed on the best way to help users access a service by authenticating them with security codes. If your service users security codes for this purpose, get in touch to share your user research findings.

If your service uses this pattern, we’d be interested to hear any insights you have on any accessibility considerations.
