Access a service with reference numbers

Also known as authenticate or identifying users

EXPERIMENTAL

This pattern is currently experimental because more research is needed to validate it.

Let users access a service in a secure way by authenticating themselves with reference numbers.

When to use this pattern

Follow this pattern if you are helping users access a service by authenticating users with reference numbers. A reference number is a simple way to let users access a service. For example when a user saves progress and needs to return to complete a service.

Services using this pattern

  • Passport renewals

Research

More research is needed on the best way to help users access a service by authenticating them with reference numbers. If your service users reference numbers for this purpose, get in touch to share your user research findings.
