When to use this pattern

Follow this pattern:

  • to draw a user’s attention to important information specific to that point in their journey
  • to allow users to acknowledge something before continuing
  • if you find that users are missing important information on a page

When not to use this pattern

Do not follow this pattern:

  • when including large amounts of content - the effectiveness of the pattern is based on its unexpected appearance so should be used sparingly

How it works

The interruption card pattern should:

  • include a heading
  • have large body text - a minimum of 24px
  • use a “Continue” button to move on to the next card or page
  • have one thing per card - if the user needs to do more than one thing, use extra cards

Services using this pattern

  • Passport renewals
  • Verify (GOV.UK)

Research

This pattern has been so successful on the passport renewals service that we have been able to dramatically reduce the amount of content on the start page.

Interrupting users’ flow in this way can increase their understanding and retention of important information.

We have seen evidence that the contrast levels of interruption cards help dyslexic users.
